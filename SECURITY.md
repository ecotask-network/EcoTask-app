# Security Policy

## Reporting a Vulnerability

We take the security of EcoTask seriously. If you discover a security vulnerability, please **do not** open a public issue.

Instead, send a detailed report to **[ecotask-network@proton.me](mailto:ecotask-network@proton.me)**.

### What to include

- Type of vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)
- Your contact information for follow-up

### What to expect

- **Acknowledgment** within 48 hours
- **Update** on the status of the report within 5 business days
- **Disclosure** coordinated with you once a fix is released

We will acknowledge reports promptly and work with you to resolve the issue. We ask that you allow us reasonable time to fix the vulnerability before disclosing it publicly.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Bug Bounty

We do not currently offer a bug bounty program, but we will publicly acknowledge responsible disclosures (with your permission).
